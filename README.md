# Idula Park Arena

A modern, responsive hospitality and recreation website designed to showcase premium accommodation, fitness, wellness, sports, and event-hosting services.

![Idula Park Arena Banner](./public/banner.jpg)

## Overview

Idula Park Arena is a premium hospitality and recreation destination offering:

* Hotel Accommodation
* Gym & Fitness Center
* Conference Hall
* Swimming Pool
* Football Pitch
* Event Hosting
* Massage & Wellness Services

This project provides a modern web presence that allows visitors to explore facilities, view galleries, learn about services, and submit booking requests through a simulated booking system.

## Features

### Public Website

## Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/idula-park-arena.git
```

### Navigate Into Project

```bash
cd idula-park-arena
```

### Run Locally

Simply open:

```bash
index.html
```

or launch a local server:

```bash
python -m http.server 8000
```

Then visit:

```text
http://localhost:8000
```
